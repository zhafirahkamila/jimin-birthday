DROP YOUR MUSIC HERE
====================

Place a file named `background.mp3` (or any name — see below) in
this folder. It will play softly when a visitor enters the site,
loop continuously, and start at ~25% volume with a smooth fade-in.

Recommended:
  - Ambient / instrumental / cinematic
  - No vocals (they compete with the on-page copy)
  - ~2-4 MB (keeps the site fast)
  - A "clean" loop point so the seam is inaudible

TO USE A DIFFERENT FILENAME OR FORMAT (ogg / m4a / wav):
  Edit index.html and change the <audio id="bg-audio" src="…"> tag
  near the closing </body>. The comment above it says "SWAP MUSIC HERE".

Free / permissive sources for royalty-free ambient music:
  - https://freemusicarchive.org  (filter: Instrumental, CC BY)
  - https://pixabay.com/music/
  - https://incompetech.com  (Kevin MacLeod — attribution required)

The rest of the site works fine with no audio file — the music
button will simply show the play icon and stay silent until tapped.
